# GitGetr AI Model

This repository contains the AI model for the GitGetr Chrome extension.

## Files
- `model.json`: Architecture and configuration details of the model.
- `weights.h5`: Trained weights of the model.
- `preprocessing.py`: Preprocessing functions for data preparation.
- `utils.py`: Utility functions used in model training or inference.
- `requirements.txt`: Dependencies needed to run the code.
- `example_usage.ipynb`: Jupyter Notebook demonstrating model usage.

## Usage
To use the model, follow the instructions provided in `example_usage.ipynb`.

## Requirements
Python 3.x
TensorFlow 2.x
NumPy
